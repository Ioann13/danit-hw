3.  -rw------- : Опишіть цей дозвіл. Як додати прапорець виконуваного файлу? : Власник файла може читати і писати в цьому файлі. для зміни прав на цей фаіл застосовуємо команду chmod 766 чи у буквенному форматі: +x, +r, +w. 
 chmod +x script.sh: Добавить право исполнения для всех пользователей.
 chmod u+r file.txt: Добавить право чтения для владельца файла.
 chmod go-w file.txt: Убрать право записи для группы и остальных пользователей.

4.Яка різниця між apt і dpkg? :dpkg є низькорівневим інструментом, який працює безпосередньо з пакетами в форматі .deb. Він встановлює, конфігурує або видаляє окремі пакети. dpkg здатний виконувати ці дії без підтримки залежностей, тому це набагато менш зручний інструмент, ніж apt.

apt – це вищорівнева програма, яка використовує dpkg для керування пакетами. Він автоматично визначає і встановлює потрібні залежності, спрощуючи процес установки або видалення пакетів. Крім цього, apt також дає змогу виконувати пошук нових пакетів, оновлення системи та забезпечує більш зручний інтерфейс для користувача.
